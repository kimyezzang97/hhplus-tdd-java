## 1주차 기본과제
___
### 기본과제 사항
- 포인트 충전, 사용에 대한 정책 추가 (잔고 부족, 최대 잔고 등)
- 주어진 4가지 기능에 대한 단위 테스트 작성
  - 포인트 조회
  - 포인트 충전
  - 포인트 사용
  - 포인트 내역 조회

### 정책사항
- 유저의 포인트는 1,000,000 포인트를 초과할 수 없다.
- 유저의 포인트는 0미만일 수 없다.
- 유저는 포인트 사용시 1,000 이상으로 사용해야 한다.
